# Dashboard Folder

This folder contains assets and functionality for the Camera Issues Dashboard.

## Files

- `dashboard.js` - Main dashboard logic and data population
- `DashBackground.jpg` - Background image for the dashboard (add your image here)

## Background Image Requirements

The background image should be:
- Format: JPG, PNG, or WebP
- Recommended size: 1920x1080 or larger
- The dashboard panels will overlay this image with glassmorphic effects
- Suggested: A scenic or technical image that fits the UDOT camera theme

## Color Scheme

The dashboard uses the app's green accent color: `#3e8e41`
- Primary green: `#3e8e41`
- Darker green: `#2a6b2f`
- Glassmorphic panels with transparency and blur effects
